# CalcProd Limpeza 🧹✨

Um aplicativo web moderno e intuitivo para o **dimensionamento de equipes de limpeza**. O CalcProd Limpeza permite calcular com precisão a quantidade de profissionais (postos de trabalho) necessários com base em diferentes ambientes, metragens e frequências de higienização.

## 🚀 Funcionalidades

- **Múltiplos Ambientes:** Adicione e gerencie vários ambientes (ex: Edifício Principal, Anexo, Área Externa) em um único relatório.
- **Cálculo Automático:** Calcula automaticamente a quantidade de postos de trabalho necessários com base na área (m²) e na frequência de limpeza selecionada.
- **Índices Personalizáveis:** Ajuste os índices de produtividade padrão para se adequarem à realidade específica do seu contrato ou local.
- **Relatório para Impressão:** Gere um relatório limpo e formatado, pronto para ser impresso ou salvo em PDF.
- **Interface Responsiva:** Design moderno construído com Tailwind CSS, adaptável a diferentes tamanhos de tela.

## 🛠️ Tecnologias Utilizadas

- [React](https://reactjs.org/) (com Hooks)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Ícones)
- TypeScript

## 💻 Como executar o projeto localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/nome-do-repositorio.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd nome-do-repositorio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. Abra o navegador no link indicado (geralmente `http://localhost:5173`).
